  • public function replace(string $fingerprint, array $options, bool $wait)
    Replace the configuration of a image
    Configuration is overwritten, not merged. Clients should first call the info method to obtain the current configuration of a image. The resulting objects metadata should be modified and then passed to the update method. $image = $lxd->images->info(null, 'b0f4faff46c9cb02db10984e2cf71c62fd539a9ab680d6fd54955671f3186087'); $image['metadata']['public'] = true; $lxd->images->replace('container-name', $image['metadata']);
